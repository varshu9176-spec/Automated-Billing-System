PROJECT TITLE  AUTOMATED BILLING SYSTEM

Project Scope

The Automated Billing System is a web-based application developed to simplify and automate the billing process for retail shops and small businesses. It enables users to manage products, customers, and billing information efficiently while reducing manual work and calculation errors.

The scope of the project includes:

User authentication through a login page.
Product management (Add, View, and Update product details).
Customer management (Add and View customer information).
Automated bill generation based on selected products and quantities.
Automatic calculation of the total bill amount.
Storage of product, customer, and billing records in a MongoDB database.
Display of reports for generated bills.
